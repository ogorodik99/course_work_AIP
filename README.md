# Курсовая работа по АИПу
# Разработка программы «Система записи к врачу»

# Разработать программу для онлайн-записи пациентов на прием к врачам медицинского учреждения.

# Требования к функциональным характеристикам:
- Ведение расписания работы врачей
- Онлайн-запись на прием через интерфейс пользователя
- Напоминание о предстоящем приеме
- Учет медицинских карт пациентов
- Управление очередью пациентов
- Формирование отчетов о работе врачей

# Требования к надежности:
- Защита персональных данных пациентов
- Резервное копирование медицинской информации
- Контроль целостности данных
- Проверка корректности временных интервалов

# Требования к составу технических средств:
- Сервер с ОС Linux/Windows Server
- Рабочие станции с веб-браузерами
- Система бесперебойного питания

# Требования к программной части:
- Язык программирования: Python/Java/PHP
- Веб-интерфейс для пациентов и администраторов
- Интеграция с СМС-рассылкой
- Система электронной очереди

# Спецификация входных и выходных данных

Перечень и требования ко входным данным:
- Расписание работы врачей
- Данные пациентов
- Запросы на запись к специалистам
- Информация о отменах и переносах записей

Перечень ограничений на входные данные:
- Время приема не должно пересекаться для одного врача
- Обязательная идентификация пациента
- Соблюдение временных интервалов между приемами

Перечень и требования к выходным данным:
- Расписание приемов на день/неделю
- Напоминания о записи
- Отчеты о посещаемости
- Статистика работы медицинского учреждения
